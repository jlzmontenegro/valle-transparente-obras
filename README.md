# Valle Transparente — Explorador de obras (no oficial)

Explorador de las obras publicadas por la Gobernación del Valle del Cauca en
https://valletransparente.valledelcauca.gov.co/ . Herramienta ciudadana derivada,
con datos extraídos de la API pública de ese portal. No es un sitio oficial.

Publicado con GitHub Pages: solo un archivo autónomo (index.html).
